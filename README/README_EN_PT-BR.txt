INFORMATION ABOUT THE ONLINE MULTIPLAYER MODE

-As online multiplayer mode has details that can not be
  so easy to find playing the game, this section has been created.

-Remember to allow the game to be played in the network by using your firewall.

-The online multiplayer mode works only on local area network (LAN).
 In order to play with players outside of your LAN, you must
 use a program to create this kind of network, such as Hamachi.

- This mode can be performed only once per connection. Therefore,please close the game after a multiplayer match and open again for a new match.

-The following step by step shows how to play the online multiplayer game:
   
1 - Player1 opens the Server Menu (Two Players -> Server);
2 - Player1 click on the "Open Server" button;
3 - Player2 opens the Client menu (Two players -> Client);
4 - Player2 passes the IP and the server port*;
5 - Player2 connects to the server (click on Connect);
6 - Player1 clicks on "Play";
7 - Player2 clicks on "Play".

* To pass the IP and the server port, the player in client mode has two options:
1 - Change IP and default port using the "Manual" setting.
2 - Change the default IP configuration and Port on config.txt, which is within the "txt" folder.

		INFORMAÇÕES SOBRE O MODO MULTIPLAYER ONLINE

-Como o modo multiplayer online possui detalhes que podem não ser
  tão faceís de descobrir jogando o jogo, essa seção foi criada.

-Não se esqueça de permitir o jogo a funcionar em rede pelo controle do seu firewall.

-O modo multiplayer online funciona apenas em rede local (LAN).
 Para que você jogue com jogadores fora de uma rede LAN, é necessário
 o uso de algum programa que crie esse tipo de rede, como o Hamachi.

- Este modo funciona apenas uma vez por conexão realizada. Isso se deve a Seek Of Peace
ser o primeiro jogo multiplayer do programador do jogo. Tendo isso em consideração,
solicitamos que fechem o jogo após a partida multiplayer e abram novamente para uma nova partida.

-O seguinte passo a passo mostra como jogar o jogo modo multiplayer online:
   
1 - JOGADOR1 abre o Menu do servidor (Dois Jogadores --> Servidor);
2 - JOGADOR1 dá um click no botão "Abrir servidor";
3 - JOGADOR2 abre o Menu do cliente (Dois jogadores --> Cliente);
4 - JOGADOR2 passa o IP e a porta do servidor*;
5 - JOGADOR2 conecta-se ao servidor (click em conectar);
6 - JOGADOR1 dá um click em jogar;
7 - JOGADOR2 dá um click em jogar.

* Para passar o IP e a porta do servidor, o jogador no modo cliente tem duas opções:
1 - Alterar o IP e a porta padrão utilizando o modo de conexão Manual;
2 - Alterar a configuração padrão do IP e da porta no arquivo config.txt, que está dentro da pasta "txt".
